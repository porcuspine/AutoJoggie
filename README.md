## Guide

"Toggler" mode turns each click into a cycle of two near-sequential clicks. Among other potential uses, this mode is for the case where a user wants to repeatedly toggle a subject, such as retracting and re-casting a fishing line in a video game.

## Disclaimer

The use of an autoclicker is explicitly banned in the terms of service of most online games. This is a personal accessibility tool for the automation of private tasks; do not use this tool to cheat or otherwise gain an unfair advantage in online games.
